ABOUT

Danix is a passionate Java developer in their second year of college, with a strong foundation in Spring Boot, Spring Security, and various Java frameworks. They have a keen interest in creating web applications and have developed projects like a high-level library management system. Danix is also exploring frontend technologies, including React and Tailwind CSS, to enhance their full-stack development skills. They are committed to continuous learning, currently focusing on Data Structures and Algorithms, and aim to turn their projects into real-world products. Additionally, Danix shares their coding journey and insights through an Instagram channel dedicated to technology and programming.

The primary goal of ShelfSpace is to streamline library management processes, making it easier for librarians to manage books and for users to access library resources. By automating routine tasks and providing useful features like notifications and search capabilities, ShelfSpace aims to enhance the overall library experience.
